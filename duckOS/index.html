duckOS Demo

duckOS on GitHub | Made with v86

This demo runs duckOS on an emulated x86 CPU in your browser.
Thus, it will be slower than emulating or virtualizing it through conventional methods.
The build of duckOS in this demo was last updated on: Jan 2, 2023.


The disk image is streamed in 5KB chunks from the server as needed; opening applications and files for the first time may be slow. The orange activity light above indicates disk activity.
